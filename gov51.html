Course Description

This course introduces students to quantitative methods for analyzing political data. We cover four main pillars: causality (how do we know if X causes Y?), measurement (how do we summarize and visualize data?), prediction (how do we use data to forecast outcomes?), and inference (how confident should we be in our conclusions?).

Throughout the course, students learn to program in R using real political science datasets. By the end, you will be able to critically evaluate empirical claims, conduct your own data analyses, and communicate findings effectively.

Learning Objectives

By the end of this course, you will be able to:

  • Distinguish causal claims from correlational ones and identify threats to causal inference
  • Summarize and visualize data using appropriate statistics and graphics
  • Use linear regression to make predictions and quantify relationships
  • Construct confidence intervals and conduct hypothesis tests
  • Write R code to load, manipulate, analyze, and visualize data
  • Communicate statistical findings clearly and honestly

Required Text

  • Imai, Kosuke and Nora Webb Williams. Quantitative Social Science: An Introduction in tidyverse. Princeton University Press, 2022. Publisher | Amazon

This is the tidyverse edition of Imai's popular QSS textbook. All R code uses modern tidyverse syntax (dplyr, ggplot2, etc.).

Course Policies

Academic Integrity

You are expected to complete all work honestly and independently unless collaboration is explicitly permitted. On problem sets, you may discuss approaches with classmates, but you must write your own solutions and code. List all collaborators on your submission.

AI Policy

Do not use AI assistants (ChatGPT, Claude, GitHub Copilot, etc.) on problem sets or exams. The goal of this course is for you to learn to think with data. Using AI to generate answers defeats that purpose and will leave you unprepared for exams, which are completed in-class without AI assistance.

Accessibility

If you have a disability or condition that may affect your learning, please contact the Disability Access Office and speak with me so we can make appropriate accommodations.

Getting Help

If you're struggling, please reach out early. Come to office hours, attend TF sections, and use the course discussion board. Learning statistics is hard—confusion is normal and expected.
